# Dynamic Form Builder

**[Live Demo →](https://hamdi-karim.github.io/Form-Builder/)**

## Features

- **Build custom forms** — Add fields with labels, placeholders, and choose between text or password inputs
- **Live preview** — Switch between Builder and Generated Form views to see your form in action
- **Add & remove fields** — Easily add new fields or remove existing ones with confirmation
- **Form validation** — Can't save until all fields have labels filled in
- **Submit & success page** — Submit the generated form and see a success message
- **Fully tested** — Includes Cypress E2E tests covering all main functionality
